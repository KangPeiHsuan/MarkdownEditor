## 🎯 支援的 Markdown 語法

### 標題
- `# H1 標題` - 24pt 粗體
- `## H2 標題` - 20pt 粗體  
- `### H3 標題` - 18pt 粗體
- `#### H4 標題` - 16pt 粗體

### 文字格式
- `**粗體文字**` - 粗體樣式
- `*斜體文字*` - 斜體樣式
- `++底線文字++` - 底線樣式

### 其他
- `- 列表項目` 或 `* 列表項目` - 項目符號列表
- `> 引用文字` - 引用格式（斜體）
